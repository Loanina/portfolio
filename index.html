My Projects

Interactive Projector Games at KidsJumpTech

From September to January 2024, I developed 4 interactive projector games for KidsJumpTech. The projector transmits an image to the wall and responds to touch, allowing players to interact with virtual objects in real time. All games are optimized to work on weak devices. Each game has single and two-player modes, which makes them suitable for both individual and competitive gameplay.

Archery

A game in which players compete with precision by shooting at targets with an arrow. Realistic gameplay and rich visual effects create an exciting atmosphere.

Soccer target

A game in which you need to shoot down targets with soccer balls. The simple and dynamic mechanics make it suitable for both single-player games and two-person competitions. Perfectly develops accuracy and coordination.

Hockey target

An energetic game where participants shoot down targets with hockey pucks. A great choice for kids activities and training.

Build a snowman faster

A winter game where participants catch snowballs to build a snowman faster. The cozy atmosphere and the excitement of the competition make this game ideal for winter entertainment.

Pet projects

Minecraft Arcanoid

An arkanoid game optimized for Android. The player controls the platform, hits the ball and destroys the blocks, passing the levels with increasing difficulty. The game includes elements inspired by Minecraft, such as stylized design and unique power-ups that add variety to the gameplay.

Mobile Simulator: Joystick & Swipe Controls Implementation

A compact project showcasing the implementation of intuitive joystick and swipe controls for a mobile simulator. This demo focuses on smooth and responsive mechanics, designed to enhance the user experience in mobile gaming.
